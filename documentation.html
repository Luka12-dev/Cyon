Cyon Language Documentation

Complete guide to building high-performance applications with Cyon

Overview

Cyon is a modern systems programming language that compiles to native machine code via C. It combines the performance of low-level languages with the ergonomics of high-level languages, making it ideal for systems programming and performance-critical applications.

Installation

Prerequisites

Before installing Cyon, ensure you have:

  • Python 3.8 or higher
  • GCC or Clang compiler
  • Make (for building the runtime)

Installation Steps

# Clone the repository git clone https://github.com/Luka12-dev/Cyon.git cd Cyon # Initialize project structure python compiler.py init # Build the runtime library cd core/runtime make cd ../.. # Verify installation python compiler.py info
Note

On Windows, you may need to add Cyon to your PATH or use the provided batch scripts in the bat/ directory.

Quick Start

Get up and running with Cyon in minutes:

1. Create Your First Program

func main() { print("Hello, Cyon!"); }

2. Run Your Program

python compiler.py run hello.cyon

3. Build an Executable

python compiler.py build hello.cyon ./build/hello

Your First Program

Let's create a simple calculator program to demonstrate Cyon's syntax:

func main() { print("Simple Calculator"); int a = input_int("Enter first number: "); int b = input_int("Enter second number: "); int sum = a + b; int diff = a - b; int product = a * b; print("Sum:", sum); print("Difference:", diff); print("Product:", product); }

This program demonstrates:

  • Function definition with func
  • Variable declarations with type annotations
  • User input with input_int()
  • Arithmetic operations
  • Output with print()

Syntax

Basic Syntax Rules

  • Statements end with semicolons (;)
  • Blocks are delimited by curly braces ({ })
  • Comments use // for single-line
  • Case-sensitive identifiers
  • Colons (:) are used after conditions in if/while/for statements

Control Flow

If Statement

func main() { int age = 18; if age >= 18: print("Adult"); }
Syntax Note

Notice the colon (:) after the condition. This is required in Cyon's if statements. The compiler currently supports single-statement if bodies without explicit else clauses.

While Loop

func main() { int count = 0; while count < 5: print(count); count = count + 1; }

Complete Control Flow Example

func main() { print("Number Checker"); int num = input_int("Enter a number: "); if num > 0: print("Positive number"); if num == 0: print("Zero"); if num < 0: print("Negative number"); }

Functions

Function Definition

// Basic function with no parameters func greet() { print("Hello, World!"); } func main() { greet(); }
Current Limitation

The current version of Cyon compiler supports function definitions but has limited support for parameters and return values. Functions primarily work with global state or side effects (like printing).

Arrays

Arrays in Cyon are currently in development. The runtime includes comprehensive array support that will be exposed in future compiler versions.

Coming Soon

Array support is being actively developed. The runtime library already includes functions like cyon_array_new(), cyon_array_push(), and more. These will be accessible from Cyon code in upcoming releases.

CLI Commands

Cyon provides a comprehensive command-line interface through compiler.py:

  • python compiler.py run <file.cyon>
    Compile and immediately execute a Cyon program
  • python compiler.py build <file.cyon>
    Build an executable in the build/ directory
  • python compiler.py build <file.cyon> --target <platform>
    Build for specific platform: native, windows, linux, or macos
  • python compiler.py init
    Initialize a new Cyon project with required directory structure
  • python compiler.py clean
    Remove build artifacts from the build/ directory
  • python compiler.py info
    Display system information and compiler configuration

Compilation Process

Understanding how Cyon transforms your code into native executables:

Compilation Pipeline

.cyon Source Code ‚Üì [Lexer] ‚Üí Tokens ‚Üì [Parser] ‚Üí Abstract Syntax Tree (AST) ‚Üì [Optimizer] ‚Üí Optimized AST ‚Üì [Code Generator] ‚Üí C Source Code ‚Üì [GCC/Clang] ‚Üí Native Machine Code ‚Üì Executable Binary

Example 4: Loop Demo

Example 5: Fibonacci Sequence

func main() { print("Fibonacci Sequence"); int n = 10; int prev = 0; int curr = 1; int i = 0; while i <= n: if i == 0: print("F(0) = 0"); if i == 1: print("F(1) = 1"); if i > 1: int temp = curr; curr = curr + prev; prev = temp; print("F(", i, ") =", curr); i = i + 1; }
